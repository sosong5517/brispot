## BRISPOST

### Requirements

1. Install NVM (Node Version Manager)
2. Install Nodejs, `nvm install latest`

### Installation

`npm i`

----

## Running

**After clone**

```sh


$ npm install
```

**Starting the application**
node dbMigrations
```sh
$ npm start
```

or

```sh
$ node index.js
```

---


### Environment Setup



Environment Variables:

1. Copy `.envexample` to `.env`
2. Never commit `.env` file!
